Embed your twitter account on your site and grow your visitors. Show twitter widgets or short tweets on your posts.

== Description ==

Twitter tweets and feeds makes it easy for the users to show their twitter feeds into their site. Customizable embedded Tweet colors from the WordPress admin interface. Customizable widget. You can choose a widget theme color, enable/ disable thumbnail, enable/disable touch, customize height of the widget, add follow button and many more.

You can embed your twitter feeds into any posts using shortcodes. Easily customizable. Clean and elegants. No Ads !!

**Features:**

* Clean UI
* Themes
* Customizable widget settings
* Shortcodes
* Single Post on any Post
* Widget Images Popup
* Caching Enabled
* Touch Scrollable for Widgets
* Well Documented
* No Ads !!

Documentation on https://deepenbajracharya.com.np/documentation/twitter/1.0/

Contribute to https://github.com/techies23/twitter-using-twitterapi and help me improve. I am open for suggestions. Please post on the support forum. PRO version with one click oAuth2.0 and various features will be available soon.

== Installation ==

1. Search for "Tweet using TwitterAPI" in the search section.
2. Install plugin.
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

### How to Connect my Twitter Account

Goto Settings page in settings -> Twitter Settings and configure your keys here.

### How to Generate Keys

Goto https://apps.twitter.com/ to generate the keys. Create a New app and generate keys from here. https://apps.twitter.com/app/new

### More FAQ

Follow here https://deepenbajracharya.com.np/documentation/twitter/1.0/#faq
